# SISDEL - Sistemas de Desarrollo en Línea

Sitio web oficial de SISDEL. Automatización inteligente para facilitar el crecimiento de las PYMEs.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **Tailwind CSS**: Estilizado moderno y responsivo.
- **JavaScript**: Animaciones interactivas utilizando `IntersectionObserver`.
- **Google Fonts**: Tipografía Inter para una lectura premium.
- **Material Symbols**: Iconografía moderna de Google.

## Características

- Diseño responsivo orientado a la conversión.
- Animaciones de scroll suaves.
- Estética moderna con efectos de glassmorphism.
- Optimizado para SEO.
